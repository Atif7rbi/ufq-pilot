# Ufq Pilot

Operational pilot system for شركة أفق السكنية للتطوير العقاري.

## Structure

- `backend/` — Laravel 12 API
- `frontend/` — Next.js application
- `docs/` — Project documentation

## Core Environment

- PHP 8.2
- Laravel 12
- PostgreSQL
- Node.js 20
- Next.js
- TypeScript
- Tailwind CSS

## Project Policy

This repository is independent from NexusOS Core.

It is a production-ready operational pilot built specifically for Ufq Housing Development Company.
